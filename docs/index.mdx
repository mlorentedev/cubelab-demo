# CubeLab Demo

Bienvenido al laboratorio distribuido CubeLab Demo: un proyecto real para aprender y aplicar DevOps con hardware físico usando Raspberry Pi + Jetson Nano + Kubernetes + Python.

---

## 🧱 Descripción general

Este proyecto simula una arquitectura cloud usando MinIO, Kubernetes y FastAPI en un clúster físico.

- Raspberry Pi actúa como Load Balancer/API Gateway
- Jetson Nano A sirve como almacenamiento (MinIO)
- Jetson Nano B procesa imágenes (detección y thumbnails)
- Todo se orquesta con Kubernetes (K3s)
